

## What is the difference between var, let, and const?

`var`, `let`, and `const` are keywords used to declare variables in JavaScript.

- **var** is function-scoped and can be redeclared and reassigned.
- **let** is block-scoped and can be reassigned but cannot be redeclared in the same scope.
- **const** is also block-scoped but its value cannot be reassigned after declaration.

In modern JavaScript, developers mostly use **let** and **const** instead of **var**.

## What is the spread operator (...)?

The **spread operator (`...`)** is used to expand elements of an array or properties of an object.  
It is commonly used to copy arrays, merge arrays or objects, and pass multiple values easily.

---

## What is the difference between map(), filter(), and forEach()?

`map()`, `filter()`, and `forEach()` are array methods used to iterate over array elements.

- **map()** creates and returns a new array after applying a function to each element.
- **filter()** creates and returns a new array containing elements that match a specific condition.
- **forEach()** executes a function for each element but does not return a new array.

---

## What is an arrow function?

An **arrow function** is a shorter syntax for writing functions in JavaScript.  
It uses the `=>` symbol and provides a more concise way to write functions compared to traditional function expressions.

---

## What are template literals?

**Template literals** are a way to create strings in JavaScript using backticks (``).  
They allow embedding variables and expressions inside a string using the `${}` syntax, making string formatting easier and more readable.
